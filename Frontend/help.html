Bestellvorgang

1. Warenkorb

Wenn Sie Ihr gewünschtes Produkt ausgewählt haben, können Sie dieses "in den Warenkorb" legen. Die Anzahl der Produkte im Warenkorb können Sie jederzeit "ändern" oder aus dem Warenkorb "löschen". Wenn Sie die Produkte im Warenkorb kaufen wollen, klicken Sie den Button [Zur Kasse].

2. Login / Registrierung

Sie können "als Gast" fortfahren (Bestellung ist nicht möglich) oder mit Ihren Benutzerdaten einloggen / registrieren.

3. Adressdaten überprüfen

Prüfen Sie Ihre Rechnungsadresse bzw. Lieferadresse.

4. Zahlungsart auswählen

Wählen Sie Ihre bevorzugte Zahlungsart.

5. Bestellung abschließen

Bitte bestätigen Sie die "AGB" und Ihr "Widerrufsrecht". Auf der Bestätigungsseite werden noch einmal alle Details aufgelistet. Sie haben hier jederzeit die Möglichkeit, Korrekturen vorzunehmen. Durch Anklicken des Buttons „zahlungspflichtig bestellen“ schließen Sie den Bestellvorgang ab.

Zahlungsarten

Für Käufer aus Österreich

1. Rechnung: Sie erhalten Ihre Rechnung per Post oder per E-Mail mit Kontodaten zur Überweisung
2. VISA / Verified by VISA*
3. Master Card / Secure Code
4. EPS Online-Überweisung
5. Klarna (Sofortüberweisung)
6. Paypal

Für Käufer aus dem Ausland

Alle Zahlungsmittel außer Rechnung.
